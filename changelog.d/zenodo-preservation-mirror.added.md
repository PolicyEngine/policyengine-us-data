Added Zenodo preservation-mirror foundation. New `policyengine_us_data/utils/zenodo_client.py` wraps the Zenodo REST API with a typed `create_and_publish_deposit()` helper that creates a deposit, uploads files, publishes, and returns the version + concept DOIs. Env-var-gated (`ZENODO_ACCESS_TOKEN` required; `ZENODO_BASE_URL` optional for sandbox) so unconfigured environments no-op cleanly. Extended `build_release_manifest()` with `preservation_mirrors_by_artifact` and `preservation_dois` kwargs that populate the fields introduced in policyengine.py PR #317 on the emitted manifest JSON. Modal-build wiring deferred to a follow-up PR that requires a production Zenodo access token. 11 new zenodo-client tests + 3 new release-manifest tests. Addresses #810.
